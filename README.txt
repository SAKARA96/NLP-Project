There are 6 main files for the project namely:
1. sentiment2.py
2. numericclassification.py
3. bagofwords_dict.py
4. score.py
5. totaldata.xlsx
6. train_sent_final.json

To run the project place all the 6 files in the same directory and execute "numericclassification.py"

Note: numericclassification.py generates train.csv on execution.
